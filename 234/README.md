# 💕 恋爱234天纪念 - 躲弹幕小游戏 💕

一个浪漫的前端躲弹幕小游戏，专门为纪念234天的恋爱而制作。

## 🎮 游戏特点

- **移动端优化**：专为手机触摸屏设计的游戏体验
- **触摸跟随**：手指触摸屏幕，爱心角色会平滑跟随移动
- **渐进式难度**：游戏难度随时间逐渐增加，从简单到极限模式
- **浪漫主题**：粉色渐变背景、爱心角色、浪漫的视觉效果
- **流畅体验**：使用Canvas绘制，60FPS流畅动画
- **记录系统**：自动保存最高记录
- **全屏适配**：完美适配各种手机屏幕尺寸和方向

## 🕹️ 游戏说明

### 操作方式
- **触摸屏幕**：用手指触摸屏幕任意位置，爱心角色会跟随你的手指移动
- **平滑跟随**：角色会平滑地移动到你手指的位置
- **躲避障碍**：躲避从上方掉落的各种形状的障碍物
- **生存挑战**：尽可能存活更长时间

### 难度等级
- **0-10秒**：简单模式
- **10-25秒**：普通模式
- **25-45秒**：困难模式
- **45-70秒**：非常困难模式
- **70秒+**：极限模式

## 🚀 如何运行

### 方法1：直接在手机浏览器打开
1. 将项目文件上传到服务器或使用GitHub Pages托管
2. 在手机浏览器中访问网址
3. 点击开始游戏即可畅玩

### 方法2：本地测试
1. 使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve
   ```
2. 在手机浏览器中访问电脑IP地址和端口号（如：http://192.168.1.100:8000）
3. 也可在电脑浏览器中访问进行测试

### 推荐浏览器
- iOS Safari
- Android Chrome
- 微信内置浏览器
- 其他支持触摸事件的现代移动浏览器

## 💖 游戏特色

- **心形角色**：玩家角色是一个可爱的粉色爱心，触摸时会变大
- **触摸反馈**：触摸时爱心会发光，并显示移动指示线
- **多样化障碍**：圆形、方形、三角形等不同形状的旋转障碍物
- **粒子效果**：爆炸和轨迹粒子特效
- **背景动画**：漂浮的爱心背景，营造浪漫氛围
- **平滑操控**：角色平滑跟随手指，留下美丽的轨迹
- **视觉提示**：实时显示生存时间、难度等级、躲避次数
