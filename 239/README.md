# Draw to Move - 恋爱239天纪念 💕

一个浪漫的画线引导小游戏，纪念我们在一起的239天！

## 游戏玩法 🎮

1. **画出路径** ✏️
   - 用手指在屏幕上画出一条路径
   - 路径会显示为粉红色的线条

2. **收集爱心** 💖
   - 引导小球沿着你画的路径移动
   - 收集关卡中的所有小爱心

3. **到达目标** 🎯
   - 让小球最终到达目标点
   - 只有收集完所有爱心才能完成关卡

4. **避开障碍** ⚠️
   - 小心不要碰到粉红色的障碍物
   - 碰到障碍会重新开始关卡

## 游戏特色 ✨

- 🎨 **自由绘制**：随心所欲画出任何路径
- 💕 **浪漫主题**：充满爱意的粉红色界面
- 📱 **移动优化**：专为手机触摸屏设计
- 🎮 **10个关卡**：从简单到极限，层层挑战
- ❤️ **239天纪念**：每一关都是我们爱情的见证

## 操作说明 📋

- **画路径**：用手指在屏幕上滑动
- **清除路径**：点击"清除路径"按钮
- **开始移动**：画好路径后点击"开始移动"
- **重新开始**：任何时候都可以重新开始当前关卡
