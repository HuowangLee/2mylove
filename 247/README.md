# 第247天 · 倾斜平衡塔 💗

## 游戏简介
纪念我们恋爱的第247天，用心形积木建造属于我们的爱情高塔！

## 玩法说明
- 🎮 **倾斜控制**：左右倾斜手机控制底部平台的角度
- 💗 **心形积木**：彩色心形积木会从上方缓缓落下
- 🏗️ **堆叠挑战**：通过调整平台角度，让积木稳稳地落在塔顶
- 📱 **触摸辅助**：点击屏幕可以提前掉落下一块积木
- ⚖️ **保持平衡**：保持塔的平衡，不要让积木掉出屏幕

## 操作方式
1. **重力感应**（推荐）：
   - 点击"开启重力感应"授予权限
   - 左右倾斜手机控制平台
   
2. **触摸控制**（备用）：
   - 触摸屏幕左侧/右侧控制平台倾斜
   
3. **校准功能**：
   - 点击"校准倾斜"将当前角度设为中立位置

## 游戏特色
- 🎨 梦幻紫色渐变背景
- 💖 真实物理引擎模拟（Matter.js）
- 🌟 流畅的倾斜控制体验
- 📊 实时显示得分和高度
- 🎯 适配移动端触摸和重力感应

## 技术实现
- **物理引擎**：Matter.js
- **设备感应**：DeviceOrientation API
- **渲染方式**：Canvas 2D
- **移动适配**：响应式设计

## 游戏目标
建造尽可能高的爱情高塔，挑战最高纪录！
