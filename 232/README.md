# 🎹 黑白键游戏

## ❤️ 恋爱第 232 天纪念 ❤️

这是一个专为移动端设计的黑白键音乐游戏，纪念我们在一起的美好时光。

## 🎮 游戏玩法

- 黑色方块会从屏幕顶部不断向下掉落
- 当黑色方块到达屏幕底部的**虚线区域**时，快速点击它
- **不要点击白色方块**，否则游戏结束
- 如果黑色方块掉到底部未被点击，游戏也会结束

## ✨ 游戏特色

- 📱 **移动端优化** - 完美适配各种手机屏幕
- 🎵 **音效系统** - 每次点击都有悦耳的音符声
- 🔥 **连击系统** - 连续点击正确，显示连击数
- ⚡ **难度递增** - 分数越高，方块掉落速度越快
- 💝 **浪漫元素** - 粉色主题、爱心特效、温馨寄语
- 🏆 **本地记录** - 自动保存最高分

## 🎯 目标

达到 **232 分**即可通关，解锁特别的爱情寄语！
