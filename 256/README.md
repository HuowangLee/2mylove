# 💕 二进制之爱 - 恋爱256天纪念 💕

## 🎮 游戏简介

《二进制之爱》是一款为纪念恋爱256天而特别制作的2048风格数字合并游戏。256是2的8次方（2⁸），象征着从最简单的"1"开始，经过8次翻倍累积，变得美好而强大——就像我们的爱情！

## ✨ 游戏特色

- 🎯 **目标明确**：通过合并2的幂次数字，最终达到256
- 📱 **移动优化**：完美适配手机端触摸滑动操作
- 🎨 **8位色彩**：每个数字都有独特的渐变配色
- 💖 **爱情主题**：融入浪漫元素的视觉设计
- 🏆 **最高分记录**：自动保存你的最佳成绩
- ✨ **特殊动画**：达到256时的炫酷庆祝效果

## 🎲 游戏玩法

1. **移动方块**
   - 📱 手机端：在游戏区域上滑动（上/下/左/右）
   - ⌨️ 电脑端：使用方向键（↑/↓/←/→）

2. **合并规则**
   - 相同数字的方块会合并成它们的和
   - 每次移动后会随机生成一个新方块（2或4）
   - 合并路径：2 → 4 → 8 → 16 → 32 → 64 → 128 → **256**

3. **胜利条件**
   - 成功合并出256即可通关！
   - 达成后仍可继续游戏挑战更高分数

4. **失败条件**
   - 当棋盘填满且无法进行任何合并时游戏结束

## 🎨 数字配色

每个数字都采用独特的渐变色彩：

- **2**: 米白色 → 淡黄色
- **4**: 淡黄色 → 浅橙色
- **8**: 浅橙色 → 深橙色
- **16**: 深橙色 → 珊瑚红
- **32**: 珊瑚红 → 火红色
- **64**: 火红色 → 金黄色（发光）
- **128**: 金黄色（强烈发光）
- **256**: 粉红 → 紫红 → 金色（旋转发光）✨

## 💡 游戏技巧

1. **角落策略**：将最大的数字放在一个角落
2. **保持秩序**：尽量让数字按大小排列
3. **避免分散**：不要让大数字四处分散
4. **预判移动**：思考每次移动的后果
5. **保持冷静**：局面混乱时不要慌张
