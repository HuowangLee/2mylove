# 💕 爱心方块 - 恋爱260天纪念 💕

> 一个为纪念恋爱 260 天而制作的经典俄罗斯方块游戏（Tetris），在方块的堆叠中感受我们的爱情累积！

## 🎯 游戏目标

控制下落的方块，通过旋转和移动，将方块排列整齐，消除完整的行，尽可能获得高分！

## 🎮 游戏玩法

### 操作控制

**💻 PC端（键盘）：**
- `← →` 左右移动方块
- `↑` 旋转方块
- `↓` 加速下落
- `空格` 快速落地

**📱 移动端（触摸）：**
- 使用屏幕上的按钮控制方块

### 游戏规则

1. **方块下落**：7种不同形状的方块会从顶部下落
2. **旋转移动**：使用控制键旋转和移动方块
3. **消除行**：当一行被方块完全填满时，该行会被消除
4. **得分计算**：
   - 单行消除：100分 × 等级
   - 双行消除：300分 × 等级
   - 三行消除：500分 × 等级
   - 四行消除（Tetris）：800分 × 等级
5. **等级提升**：每消除10行，等级提升1级，下落速度加快
6. **游戏结束**：当方块堆到顶部无法再生成新方块时，游戏结束

## ✨ 游戏特色

### 7种经典方块

- 🟦 **I型**：直线型（蓝色）- 最适合消四行
- 🟨 **O型**：正方形（黄色）- 最稳定的方块
- 🟪 **T型**：T字型（紫色）- 最灵活的方块
- 🟩 **S型**：S字型（绿色）- 需要技巧的方块
- 🟥 **Z型**：Z字型（红色）- S型的镜像
- 🟦 **J型**：J字型（深蓝）- 适合填补角落
- 🟧 **L型**：L字型（橙色）- J型的镜像

### 游戏功能

- **下一个预览**：显示下一个即将下落的方块
- **等级系统**：难度逐渐递增，挑战性十足
- **实时统计**：分数、等级、消除行数实时显示
- **暂停功能**：支持暂停和继续游戏
- **响应式设计**：完美适配PC和移动设备

### 视觉效果

- 🎨 彩色方块设计
- ✨ 方块高光效果
- 📊 清晰的网格线
- 🌈 渐变背景

## 🏆 游戏技巧

1. **留缝隙等I型**：为消四行留出空间，等待I型方块
2. **优先消除**：尽量保持方块高度平衡，避免堆得太高
3. **提前规划**：观察"下一个"预览，提前计划放置位置
4. **快速落地**：熟练后使用空格键快速落地，提高效率
5. **旋转技巧**：合理使用旋转功能，充分利用空间

## 📊 等级系统

- **1-3级**：入门阶段，速度较慢
- **4-6级**：进阶阶段，速度适中
- **7-9级**：高手阶段，速度较快
- **10级+**：大师阶段，极限挑战

## 🚀 如何运行

1. 将 `260` 文件夹放置在您的 Web 服务器或本地开发环境中
2. 在浏览器中打开 `260/index.html` 文件
3. 点击"开始游戏"按钮开始挑战
4. 使用键盘或屏幕按钮控制方块！
