# 💕 抛物线投爱心 - 243天纪念版

## 🎮 游戏简介

这是一个为纪念恋爱243天而特别制作的抛物线投掷小游戏。通过拖拉操作将爱心投递到目标位置，穿越各种障碍，完成充满挑战的关卡！

## ✨ 游戏特色

- 🎯 **物理抛物线机制**：真实的重力和抛物线轨迹
- 📱 **移动端优化**：完美支持触摸屏操作
- 🎨 **浪漫主题**：粉色系界面，爱心元素贯穿始终
- 🏆 **10个精心设计的关卡**：难度递增，挑战性十足
- ⭐ **星级评分系统**：根据剩余爱心数量评定星级
- 🎯 **移动目标**：部分关卡有移动的目标，增加难度
- 🚧 **多样障碍物**：墙壁、天花板等各种障碍考验你的精准度

## 🎯 玩法说明

### 基础操作
1. 点击/触摸发射器（左下角的大爱心）
2. 向相反方向拖动以调整角度和力度
3. 松开手指，爱心就会按照抛物线轨迹飞出
4. 击中目标区域即可过关

### 游戏目标
- 每关有3颗爱心（3次机会）
- 使用更少的爱心过关可获得更高星级
- 完成所有10个关卡即可通关

### 关卡特色
- **关卡1-3**：基础练习，熟悉操作
- **关卡4**：首次遇到移动目标
- **关卡5-6**：复杂的障碍布局
- **关卡7-8**：狭窄通道，考验精准度
- **关卡9-10**：终极挑战，移动目标+复杂迷宫

## 🎨 技术特点

- 纯前端实现（HTML5 + CSS3 + JavaScript）
- Canvas 2D 渲染
- 物理引擎模拟
- 响应式设计
- 触摸事件优化
- 粒子效果系统

## 💝 特别意义

游戏中的数字"243"代表着你们在一起的243天，每一关都代表着你们爱情路上的一个里程碑。通过游戏的方式，将爱心投递给对方，象征着爱的传递永不停歇。

## 🚀 如何开始

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始游戏"按钮
3. 享受游戏，传递你的爱！

## 📱 最佳体验

- 推荐使用移动设备（手机/平板）体验最佳触摸操作
- 也支持电脑端使用鼠标操作
- 建议竖屏游玩，获得最佳视觉效果
