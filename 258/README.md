# 💕 爱心配对 - 恋爱258天纪念 💕

> 一个为纪念恋爱 258 天而制作的经典翻牌配对记忆游戏，考验你的记忆力和专注力！

## 🎯 游戏目标

翻开所有卡片，找到相同的爱心配对，用最少的步数和最短的时间完成游戏！

## 🎮 游戏玩法

1. **点击翻牌**：点击卡片翻开，显示隐藏的爱心图案
2. **寻找配对**：每次可以翻开两张卡片
3. **配对成功**：如果两张卡片图案相同，它们将保持翻开状态
4. **配对失败**：如果图案不同，卡片会自动翻回背面
5. **完成游戏**：找到所有配对即可获胜

## ✨ 游戏特色

- **三种难度**：
  - 🟢 简单模式：3×2 (6张卡片，3对配对)
  - 🟡 中等模式：4×4 (16张卡片，8对配对)
  - 🔴 困难模式：4×5 (20张卡片，10对配对)

- **丰富的爱心图案**：20种不同的爱心emoji，每局随机选择

- **提示功能**：每局游戏提供3次提示机会，帮助你找到配对

- **统计追踪**：
  - 📊 步数统计
  - ⏱️ 时间计时
  - 🎯 准确率计算

- **精美动画**：
  - 流畅的翻牌3D动画
  - 配对成功的脉冲效果
  - 胜利时的烟花庆祝

- **移动端优化**：
  - 完美适配各种屏幕尺寸
  - 触摸操作流畅
  - 响应式布局设计

## 🚀 如何运行

1. 将 `258` 文件夹放置在您的 Web 服务器或本地开发环境中
2. 在浏览器中打开 `258/index.html` 文件即可开始游戏
3. 选择难度，点击"新游戏"开始挑战！

## 📱 最佳体验

- 支持所有现代浏览器（Chrome、Safari、Firefox、Edge 等）
- 移动端和PC端均可流畅运行
- 建议使用触摸设备获得更佳体验

2. **系统化探索**：可以按行或按列有序地翻开卡片
3. **善用提示**：在困难时使用提示功能
