# 💕 爱心打砖块 - 恋爱259天纪念 💕

> 一个为纪念恋爱 259 天而制作的经典打砖块游戏（Breakout），用爱心挡板接球，打碎所有彩色砖块！

## 🎯 游戏目标

控制挡板接住弹跳的球，打碎屏幕上所有的爱心砖块，完成关卡挑战！

## 🎮 游戏玩法

### 基本操作
- **💻 PC端**：鼠标左右移动控制挡板位置
- **📱 移动端**：触摸屏幕左右滑动控制挡板
- **🚀 发射球**：游戏开始时点击屏幕发射球

### 游戏规则
1. **接球**：用挡板接住球，不要让球掉落
2. **打砖块**：球碰到砖块会反弹并破坏砖块
3. **多重强度**：部分砖块需要击打多次才能破坏（显示数字）
4. **生命值**：你有3条生命，每次球掉落失去1条
5. **关卡进阶**：打碎所有砖块进入下一关，难度递增

### 计分系统
- 每个砖块价值：**10分 × 当前关卡数**
- 例如：第1关每个砖块10分，第2关每个砖块20分

## ✨ 游戏特色

- **渐进式难度**：
  - 每关砖块数量增加
  - 球速逐渐加快
  - 挑战性不断提升

- **多重强度砖块**：
  - 普通砖块：1次击打破坏
  - 强化砖块：2-3次击打破坏
  - 强化砖块会显示剩余强度数字

- **物理反弹**：
  - 真实的球体反弹物理
  - 击球位置影响反弹角度
  - 挡板边缘击球可以产生极限角度

- **精美视觉效果**：
  - 彩色爱心砖块
  - 发光球体和挡板
  - 流畅的动画效果

- **全平台支持**：
  - 完美适配PC和移动端
  - 响应式画布设计
  - 触摸和鼠标双重控制

## 🏆 游戏技巧

1. **预判球路**：观察球的运动轨迹，提前移动挡板
2. **边缘击球**：用挡板边缘击球可以改变角度，打到难以触及的砖块
3. **优先强化砖**：先击打需要多次破坏的砖块
4. **保持冷静**：不要慌张，稳定控制挡板位置

## 🚀 如何运行

1. 将 `259` 文件夹放置在您的 Web 服务器或本地开发环境中
2. 在浏览器中打开 `259/index.html` 文件
3. 点击"开始游戏"按钮开始挑战
4. 点击屏幕发射球，开始打砖块！
